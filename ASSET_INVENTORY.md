## DEVELOPER NOTES

### Import Schemes

#### Figma Assets (Raster Images)
```tsx
import img from "figma:asset/abc123.png"
```
- **NEVER** prefix with `./`, `../`, or any path
- Virtual module scheme, not a file path
- Correct: `figma:asset/abc123.png`
- Incorrect: `../imports/figma:asset/abc123.png`

#### SVG Imports (Vector Graphics)
```tsx
import svgPaths from "@/imports/svg-wg56ef214f"
```
- Stored in `/src/imports` directory
- Use absolute paths with `@` alias
- Do not create custom SVGs if Figma import exists

#### Component Imports
```tsx
import { ComponentName } from '@/app/components/component-name';
```
- Use `@` alias (mapped to `/src` directory)
- Never use relative paths

### Protected Files
**DO NOT CREATE OR MODIFY:**
- `/src/app/components/figma/ImageWithFallback.tsx`
- `/pnpm-lock.yaml`
